ANOTAÇÕES SOBRE O 'BOOTSTRAP':


Obs: A classe "lead" destaca o paragrafo como também pode ser usado como um paragrafo-subtitulo.

A classe text-center fixa o texto no centro.

A classe text-end fixa o texto a direita.

Use a classe form-text para uma descrição mais discreta.
